Displaying AR models on mobile app

Steps needed to successfully display models

Create a 3-D model on your favourite CAD program or find one online
Save your 3-D model as a .GLB file (This is the format needed to display the AR model)
Add your model to the web app via the form
Add a model name and description
Upload the model
Go back to the web app and the model should now be available
Click on the model to open the AR activity.....enjoy!
